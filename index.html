Use Single-Page Application

Single-Page Application (SPA) = web application that lives within a single HTML page Application views are injected into and removed from the DOM as needed Advantages are: 1. Absence of page refreshes provides more fluid and closer to native experience 2. UI is entirely created at client-side: ideal architecture for applications that work offline.

Let's Code! Use SPA

1. continu to use Employees Database Directory example 2. in index.html: remove the HTML markup inside the body tag (with the exception of the script tags). 3. inside the immediate function in app.js, define a function named renderHomeView() (right after the findByName function): function renderHomeView() { var html = "<h1>Directory</h1>" + "<input class='search-key' type='search' placeholder='Enter name'/>" + "<ul class='employee-list'></ul>"; $('body').html(html); $('.search-key').on('keyup', findByName); } 4. modify the data adapter initialization logic: when the adapter has been initialized, call the renderHomeView() function to programmatically display the Home View: var adapter = new MemoryAdapter(); adapter.initialize().done(function () { renderHomeView(); }); 5. make sure you remove the original event registration in the Event Registration section. 6. remove the click event handler for the help button (in the Event Registration section). 7. test the application.

Use Templates

Writing HTML fragments in JavaScript and programmatically inserting them into the DOM results in: 1. harder to write code 2. harder to maintain code HTML templates address this by decoupling UI (HTML markup) from code. HTML template solutions: e.g. Mustache.js, Handlebars.js, and Underscore.js.

Let's Code! Use Templates

1. continu to use Employees Database Directory example 2. in index.html: add script tag to include the handlebars.js library: <script src="lib/handlebars.js"></script> 3. in index.html: create HTML template to render the Home View. add following script tag as first child of the body tag: <script id="home-tpl" type="text/x-handlebars-template"> <div class="topcoat-navigation-bar"> <div class="topcoat-navigation-bar__item center full"> <h1 class="topcoat-navigation-bar__title">Employee Directory</h1> </div> </div> <div class="search-bar"> <input type="search" placeholder="search" class="topcoat-search-input search-key"> </div> <div class="topcoat-list__container"> <ul class="topcoat-list list employee-list"></ul> </div> </script> 4. create HTML template to render the employee list items. Add following script tag immediately after the previous one: <script id="employee-li-tpl" type="text/x-handlebars-template"> {{#.}} <li class="topcoat-list__item"> <a href="#employees/{{id}}"> <img src="assets/pics/{{pic}}"> <p>{{firstName}} {{lastName}}</p> <p>{{title}}</p> <span class="chevron"></span><span class="count">{{reports}}</span> </a> </li> {{/.}} </script> 5. add topcoat-mobile-light.css and styles.css to the head of index.html: <link href="assets/topcoat/css/topcoat-mobile-light.css" rel="stylesheet"> <link href="assets/css/styles.css" rel="stylesheet">

Minimize Reflows

Minimizing Browser Reflows

  • Reduce the number of DOM elements
  • Minimize access to the DOM
  • Update elements "offline" before reinserting into DOM
  • Avoid tweaking layout in JavaScript
